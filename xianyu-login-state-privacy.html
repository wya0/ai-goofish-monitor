闲鱼登录态采集扩展隐私权政策

版本：2026-01-13 · 本文档说明浏览器扩展“Xianyu Login State Extractor”如何收集、使用和存储数据。

单一用途

扩展唯一用途：在用户点击“提取”时，读取当前页面的登录态 Cookie、常用请求头和浏览器环境信息，生成本地 JSON 供用户复制到闲鱼监控爬虫使用。

收集的数据类型（仅在用户点击“提取”时触发）

  • 身份验证信息：当前页面的登录 Cookie/令牌。
  • 请求头：User-Agent、Accept、Accept-Language、Accept-Encoding、Referer、Sec-CH UA 系列字段、Origin、Cache-Control/Pragma、Upgrade-Insecure-Requests、Content-Type（如有）。
  • 浏览器环境信息：语言、时区、平台、hardwareConcurrency、deviceMemory、maxTouchPoints、webdriver 状态、屏幕尺寸/分辨率/色深、devicePixelRatio、User-Agent。
  • 当前页面 URL，用于生成 JSON 描述当前站点。
  • 存储数据：localStorage/sessionStorage，仅为当前页面读取，长度超过 4096 字节的键会被丢弃并标记。

数据使用方式

  • 数据仅在本地内存中拼装为 JSON，供用户手动复制；不会上传到任何服务器。
  • 不用于广告、画像、再营销或任何与上述单一用途无关的目的。

存储与传输

  • 不做持久化存储；不写入远程服务。
  • 剪贴板复制由用户主动点击触发。

权限说明

  • tabs / activeTab：仅在用户点击“提取”时访问当前标签页 URL 并注入采集脚本。
  • scripting：在当前页面注入脚本读取 navigator/screen 信息、localStorage/sessionStorage。
  • webRequest：在当前域名发起一次 fetch 读取请求头，生成 JSON，不拦截或修改流量。
  • storage：保存用户同意状态和功能开关，不存储敏感数据。
  • cookies（如使用）：读取当前页面 Cookie 以生成登录态 JSON。

用户同意与控制

  • 只有在弹窗中勾选同意并点击“提取”时才会收集上述数据；不使用则不收集。
  • 可随时关闭或卸载扩展，即停止一切收集行为。

数据共享

不与任何第三方共享或出售数据。
